 **Mystic Memory Game**

##  **Overview**

**Mystic Memory Game** is a polished, fantasy-themed memory-matching game designed to test concentration, speed, and pattern recognition. Players flip mystical symbol cards, uncover pairs, and aim to complete the grid in the least number of moves and time.

Built using **HTML, CSS, and JavaScript**, this project showcases clean UI design, smooth animations, responsive layout handling, and efficient game logic — making it perfect for portfolios, academic submissions, and front-end skill demonstration.


##  **Features**

*  16-card memory board (8 pairs)
*  Real-time timer
*  Move counter
*  Win modal with final stats
*  Restart & play-again functionality
*  Fully responsive, mobile-friendly design
*  Smooth flip animations


##  **Tech Stack**

* **HTML5** – Structure
* **CSS3** – Styling, animations, grid design
* **JavaScript** – Game logic, event handling, timer


##  **Game Logic (Summary)**

* Cards are created and randomized using CSS Grid + JS.
* Each card contains a hidden mystical emoji/symbol.
* Player flips two cards → game checks for match.
* Matches stay revealed; mismatches flip back after delay.
* Timer and move count update continuously.
* When all pairs are matched, a **victory modal** displays stats.
